Aim: Network Traffic Capture using Wireshark

1. Open Wireshark software and select your active network connection.
2. Click on the Capture tab and select the Start option.
3. Open any website and log in while Wireshark is capturing.
4. After completing the action, stop the capture in Wireshark.
5. In the captured data, look for HTTP POST methods.
6. Double-click on the POST method packet.
7. You will be able to view the username and password that were used during login (if the connection was not encrypted).
