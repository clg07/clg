
Practical No. 5 – Persistent Cross-Site Scripting (XSS) Attack

--- Setup Instructions ---

1. Install XAMPP
2. Download DVWA and extract it.
3. Move the DVWA folder to: C:\xampp\htdocs\
4. Go to: C:\xampp\htdocs\dvwa\config\
5. Rename 'config.inc.php.dist' to 'config.inc.php'
6. Edit the config.inc.php file and set:
   $_DVWA['db_password'] = '';
7. Start Apache and MySQL in XAMPP
8. Open browser and go to: http://localhost/dvwa/
9. Login with:
   Username: admin
   Password: password
10. Go to "DVWA Security" → Set security level to "Low"

--- XSS Attack Execution ---

1. Navigate to: XSS (Stored)
2. Enter any name
3. In the message box, enter the payload:
   <script>alert('XSS Attack Successful!');</script>
4. Click "Sign Guestbook"
5. Reload the page to see the alert box triggered

--- Notes ---

This is a persistent (stored) XSS attack.
It affects every user who visits the guestbook page.
